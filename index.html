ZeroClick Testing Site

Here is where you can test the current version of ZeroClick to see if it works! Try to use the following commands. If the command doesn't work, then great! If it does, then that's an issue. Please open up a new issue on the official ZeroClick GitHub repo.

Commands to try

  1. F12 (typically opens the developer tools.)
  2. CTRL+SHIFT+I (typically opens the developer tools.)
  3. CTRL+SHIFT+C (typically opens the developer tools.)
  4. CTRL+SHIFT+J (typically opens the developer tools.)
  5. CTRL+U (typically opens the "view-source" page.)
If any of these commands result in the developer tools or "view-source" pages being opened, then please be sure to report it.

Using this site for development

If you create a version of ZeroClick with updated or added code, then update this template site with more commands for users to try, and any extra notes you may have. This isn't required right now, but it would make things easier. In the future, we may require this.
